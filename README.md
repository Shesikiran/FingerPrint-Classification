# FingerPrint Classification
Built a fingerprint identification system using RBF kernel based support vector machine and used a multi-class classifier to fit the trained model and achieved an accuracy of ~90%

Download the training dataset of 126 fingerprints of 21 persons (6 thumb fingerprints per person) from this link http://bit.ly/2phOMcN. The file names follow x y.bmp convention, where x is the identification number of a person, while y is a serial number of fingerprint snap for that person.
